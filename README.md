# Premium Gym Dev Skill

[![skills.sh](https://skills.sh/b/sameerizz/premium-gym-dev-skill)](https://skills.sh/sameerizz/premium-gym-dev-skill)

Mental models, anti-patterns, and one-shot directives for building high-end, dark-theme fitness websites using Next.js, Tailwind, and Framer Motion. Eliminates common SSR and mobile layout mistakes based on the BodyZone project.

## Installation

You can install this skill for your AI agents using the open-source `skills` CLI:

```bash
npx skills add sameerizz/premium-gym-dev-skill
```

## Overview

When building or refining a premium, dark-theme fitness website, apply these mental models immediately to avoid back-and-forth iteration, SSR hydration bugs, and mobile layout clashing.

Features:
- The SSR & Animation Mental Model (The Hydration Trap)
- The Mobile Layout Hierarchy
- Custom Assets vs. Generic UI
- Typography & Accessibility Precision

See the `SKILL.md` file for full guidelines.
